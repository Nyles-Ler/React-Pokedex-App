# React Pokédex App

An interactive Pokédex web application built using React and the PokéAPI. This project allows users to search, filter, and browse Generation 1 Pokémon using dynamic API-driven content rendering and responsive UI design.

---

# Features

- Pokémon API integration using PokéAPI
- Dynamic Pokémon search functionality
- Filter Pokémon by type
- Sort Pokémon entries
- Responsive card-based layout
- Real-time data rendering
- Clean and interactive UI design
- React component-based architecture

---

# Technologies Used

- React
- JavaScript
- HTML5
- CSS3
- PokéAPI

# Installation

## 1. Clone the repository

```bash
git clone https://github.com/Nyles-Ler/Pokedex-API-With-React.git
```

---

## 2. Navigate into the project folder

```bash
cd Pokedex-API-With-React
```

---

## 3. Install dependencies

```bash
npm install
```

---

## 4. Start the development server

```bash
npm start
```
